# Contributing to Medico24-ML

Thank you for your interest in contributing to the Medico24-ML module!

## How to Contribute
- Fork the repository and create your branch from `master`.
- Write clear, concise commit messages.
- Add tests for new features and bug fixes.
- Ensure code passes linting and tests before submitting a pull request.
- Document new features and update existing documentation as needed.

## Code Style
- Follow PEP8 for Python code.
- Use type hints where appropriate.
- Keep code modular and well-documented.

## Reporting Issues
- Use the issue tracker to report bugs or request features.
- Provide as much detail as possible for faster resolution.

## Community
- Be respectful and inclusive in all interactions.
- See the main Medico24 guidelines for more details.
